# Local Source Notes

## reports-holymedia
- `facebook.py`, `facebook_adset.py`, `facebook_ads.py` were used as the main seed for Meta reporting fields, auth shape, and action/video metrics mapping.
- `YANDEX.py` was used as the seed for Yandex provider scope and basic reporting expectations.
- `GA4.py` was used as a design reference for future optional Google modules.

## google-ads
- The project served as the reference for domain concepts around reporting, diagnostics, schedules, keyword management, and safe automation patterns.
- The current MCP server keeps the domain ideas but reimplements them as Python MCP tools rather than `.gs` scripts.
