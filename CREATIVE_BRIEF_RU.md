# Шаблон Быстрого Запроса Для Креативов

Используй этот формат, когда хочешь, чтобы креативы делались быстрее и с меньшим количеством уточнений.

## Короткая команда

```text
Сделай креатив:
Цель: <лиды / сообщения / трафик / узнаваемость>
Формат: <1:1 / 4:5 / 9:16 / banner>
Продукт: <что рекламируем>
Аудитория: <для кого>
Оффер: <главное предложение>
Текст на креативе: <точный текст или "без текста">
Стиль: <строго / премиум / медицинский / нативный / UGC / яркий>
Цвета: <предпочтительные цвета>
Что показать: <сюжет, объект, человек, сцена>
Чего нельзя: <запреты>
Референс: <если есть>
Количество: <сколько вариантов>
```

## Совсем быстрый вариант

```text
Сделай 3 креатива 4:5 для Meta.
Продукт: лечение варикоза в Алматы.
Цель: сообщения в WhatsApp.
Оффер: бесплатная консультация.
Стиль: медицинский, аккуратный, доверительный.
Текст: "Боли и тяжесть в ногах? Запишитесь на консультацию".
Без: треша, кликбейта, перегруза текстом.
```

## Если нужен не только визуал

Добавляй одной строкой:

```text
Дополнительно: дай заголовки, primary text и CTA.
```

## Если нужен именно рекламный комплект

Пиши так:

```text
Сделай рекламный комплект:
- 3 варианта креатива
- 5 заголовков
- 3 текста объявления
- 3 CTA
- 2 гипотезы для A/B теста
```

## Что ускоряет работу сильнее всего

- Точный формат: `1:1`, `4:5`, `9:16`
- Точный текст на изображении
- Понимание, это Meta / TikTok / лендинг / сторис
- Что обязательно сохранить или исключить
- Референс или пример "сделай в таком духе"
